# QuizForge (Geography Quiz Generator)

QuizForge is a simple Python project that generates geography quiz questions.

This project demonstrates a basic AI workflow:

Input → generation → quiz output

Currently the generator runs in mock mode for demonstration purposes.

## How to run

python quizforge.py
